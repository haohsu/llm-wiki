# 四格漫画 Skill（开源四格漫画生成方法论）

一种基于结构化提示词的 AI 四格漫画生成方法论，灵感来源于 [[jim-liu]]（宝玉）分享的四格漫画创作模式。通过精心设计的提示词模板，让 AI 图像生成工具产出富有哲理的四格漫画故事。

## 设计理念

四格漫画的核心在于**视觉隐喻 + 哲理升华**。前三格呈现问题/冲突/转折，第四格完成智慧的传达。使用对比强烈的视觉元素（推墙与门、紧握与松手、高度与花朵），让读者在简洁的画面中获得深刻感悟。

## 故事结构模板

四格漫画遵循经典的起承转合结构：

| 格 | 功能 | 要素 |
|----|------|------|
| 第一格（左上） | 问题/现状呈现 | 场景描述、人物动作和表情、对话内容、旁白（点明主题） |
| 第二格（右上） | 问题加深/冲突 | 场景描述、人物动作和表情变化、对话内容、旁白（揭示困境） |
| 第三格（左下） | 转折/觉醒 | 场景描述、人物动作和表情转变、对话内容、旁白（引导思考） |
| 第四格（右下） | 升华/智慧 | 场景描述、人物动作和表情释然、对话内容、旁白（传达哲理）、**橙色标签**（核心关键词） |

### 故事要点

- **视觉隐喻**: 使用具象场景表达抽象概念（如推墙与门、紧握与松手、高度与花朵）
- **强烈对比**: 前三格 vs 第四格形成鲜明反差
- **结尾升华**: 传达深刻哲理，不拖沓
- **简洁有力**: 每格信息密度高，不冗余

## 图像生成提示词模板

生成插画时使用 `imageGenerate` 工具，严格遵循以下提示词结构：

```
四格漫画，2×2方格布局（两行两列），简约线条艺术风格，商务插画风格。
纯白背景，黑色线条勾勒，仅使用橙色作为重点标签的强调色，橙色使用非常克制。
除木质物体可使用极少量浅棕色外，不使用灰色调。

在整个四格漫画的顶部，用黑色清晰的中文字体写上标题："[主题名称]"。

人物风格：简化但比例协调的人物造型（非火柴人）：
圆形头部，极简五官（两个点表示眼睛，一条弧线表示嘴巴，简单的发型轮廓），
身体有清晰的衬衫领子、袖子、裤子和鞋子细节。
人物占据画面较小比例（中远景）。通过不同发型区分角色：光头、短发、中长发等。

四格布局：2行2列的方格布局，每格边框为手绘风格，线条略带自然曲线感（非完全笔直）。

风格要点：干净的线条艺术，有个性的简化人物，人物在画面中比例适当，
纯白背景，仅黑色轮廓线，橙色仅用于关键标签且非常克制，
极少阴影（仅木质物体可用浅棕色），手绘美学感的略不规则边框，
商务休闲服装轮廓清晰。
```

### 涉及品牌 Logo 时的补充

```
重要：人物的胸前要显示各自的 logo（简化版本，黑色线条勾勒，
[品牌] logo 使用温暖的铁锈橙色填充）
```

### 四格详细描述模板

每格需提供：详细的场景、人物、动作、表情、对话、旁白描述。
第四格额外强调：橙色标签是整幅画中唯一的橙色元素。

## 关键参数

| 参数 | 值 | 说明 |
|------|-----|------|
| `aspect_ratio` | `16:9` | 宽屏比例，适合四格布局 |
| `quality` | `medium` | 平衡质量和速度 |
| `title` | `[主题名称] - 四格漫画` | 图片标题 |

## 工作流程

1. **输入主题** — 用户提供创作主题
2. **故事创作** — 基于主题，按四格结构创作富有哲理的故事
3. **用户确认** — 将完整的四格故事呈现给用户确认
4. **图像生成** — 使用 `imageGenerate` 工具，按提示词模板生成插画
5. **输出交付** — 返回完整的四格漫画图片

## 设计哲学

四格漫画 Skill 体现了 [[knowledge-as-code]] 趋势的核心：**将领域专家的创作方法论编码为结构化提示词**。宝玉的四格漫画之所以好看，不在于模型能力，而在于故事结构的精心设计——视觉隐喻、对比反差、哲理升华，这些创作原则被显式地写进了提示词模板中。
