# Craft Agents

- **GitHub**: https://github.com/lukilabs/craft-agents-oss
- **Stars**: 5,586
Craft Agents 是一个开源工具，用于高效与各类智能代理协作，支持无缝连接 API 和服务，提供直观的多任务处理和文档中心化工作流。

## 核心特性

- 多 Agent 协作管理
- 无缝连接 API 和服务
- 文档中心化工作流
- 多任务处理

## 与生态的关系

- 与 [[multica]] 同属 Agent 管理平台方向
- 体现了 [[harness-engineering]] 中"Agent 作为团队成员"的理念
