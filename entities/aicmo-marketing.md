# AiCMO-Marketing

- **GitHub**: https://github.com/AICMO/AiCMO-Marketing-Prompt-Collection
- **定位**: 营销和运营方向的 Agent prompt 集合

## 核心特性

按职能模块组织：
- 品牌策略
- 内容营销
- SEO
- 社媒运营
- 广告投放
- 用户增长
- 数据分析

相当于给 Agent 装了一整个营销部门的脑子。

## 与生态的关系

- 与 [[agency-agents]] 互补：agency-agents 定义营销角色，本项目提供具体营销 prompt
- 属于 [[knowledge-as-code]] 趋势在营销领域的体现
