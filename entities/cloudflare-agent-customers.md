# Cloudflare Agent Customers

- **公司**: Cloudflare
- **方向**: 让 AI Agent 成为 Cloudflare 的一等客户

## 核心能力

Agent 可以自主完成以下操作：

1. **创建 Cloudflare 账户** — Agent 自行注册，无需人类代劳
2. **开通付费订阅** — 与 [[stripe]] 合作，Agent 可自主完成支付流程
3. **注册域名** — Agent 可自主选择和注册域名
4. **获取 API Token** — 立即获取凭证部署代码

## 意义

这是 [[agents-as-customers]] 趋势的标志性事件：云厂商开始将 Agent 视为独立的客户实体，而非仅仅是人类用户的工具。Agent 不再需要人类"帮忙"开通服务，而是自己就是服务的购买者和使用者。

## 与生态的关系

- 体现了 [[agents-as-customers]] 的核心理念
- 与 [[harness-engineering]] 中"Agent 作为独立运行体"的理念一致
- Agent 部署能力闭环：写代码 → 注册域名 → 部署上线，全程无需人类介入
