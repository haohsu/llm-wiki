# taste-skill

- **GitHub**: https://github.com/Leonxlnx/taste-skill
- **Stars**: 7k+
- **定位**: 解决 Agent 生成前端千篇一律的问题

## 核心特性

提供三个可调节旋钮（1-10 级）：
1. **设计冒险度** — 控制设计的创新程度（保守 ↔ 大胆）
2. **动画强度** — 控制动画的丰富程度（静态 ↔ 动态）
3. **视觉密度** — 控制信息密度（极简 ↔ 密集）

## 使用场景

默认情况下 Agent 生成的前端往往风格雷同。taste-skill 通过调节参数让输出更具个性化和多样性。

## 与生态的关系

- 与 [[awesome-design-md]] 互补：awesome-design-md 定义"用什么"，taste-skill 定义"怎么调"
- 属于 [[knowledge-as-code]] 趋势在前端设计领域的体现
