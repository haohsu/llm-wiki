# Maigret

- **GitHub**: https://github.com/soxoj/maigret
- **Stars**: 20,870
Maigret 是一款通过用户名收集个人信息的 OSINT 工具，支持检查数千个网站并提取可用数据，无需 API 密钥。

## 核心特性

- 通过用户名跨平台信息收集
- 支持数千个网站
- 无需 API 密钥，开箱即用
- 20k+ stars，OSINT 领域热门工具

## 与生态的关系

- 与 [[ghost-track]] 同属 OSINT 工具方向
- Agent 可用于自动化情报收集流程
