# Quarkdown

- **GitHub**: https://github.com/iamgio/quarkdown
- **Stars**: 13,099
Quarkdown 是一个基于 Markdown 的现代排版系统，支持编译为书籍、论文或交互式演示，具有图灵完备的函数扩展和标准库。

## 核心特性

- Markdown 作为源格式
- 编译为书籍、论文、交互式演示
- 图灵完备的函数扩展
- 内置标准库

## 与生态的关系

- 属于 [[knowledge-as-code]] 趋势中的"Markdown 排版"方向
- 与 [[llms-txt]] 一样推动 Markdown 成为通用信息载体
- Agent 可用它自动生成格式化文档
