# codebase-to-course

- **GitHub**: https://github.com/zarazhangrui/codebase-to-course
- **定位**: 把任意代码仓库变成交互式 HTML 教程

## 核心特性

- 输入：任意代码仓库
- 输出：单页交互式 HTML 教程
- 包含：动画可视化、嵌入式测验、代码与白话对照

## 目标用户

面向那些用 AI 写代码但想搞清楚项目到底怎么跑起来的人。

## 与生态的关系

- 与 [[knowledge-as-code]] 趋势的关系：其他项目是把知识写成 .md 喂给 Agent，codebase-to-course 是反过来——把 Agent 产出的代码转化为人类可理解的教程
- 这是一个"反向"应用：Agent → Human 知识传递
