# GhostTrack

- **GitHub**: https://github.com/HunxByts/GhostTrack
- **Stars**: 12,194
GhostTrack 是一款用于追踪位置或手机号码的信息收集工具，支持 IP 追踪、电话号码追踪和社交媒体用户名追踪等功能。

## 核心特性

- IP 地址追踪
- 电话号码追踪
- 社交媒体用户名追踪
- 多种信息收集功能集成

## 与生态的关系

- 与 [[maigret]] 同属 OSINT 工具方向
- 可与 Agent 集成实现自动化追踪工作流
