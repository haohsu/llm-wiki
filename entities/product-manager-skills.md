# Product-Manager-Skills

- **GitHub**: https://github.com/deanpeters/Product-Manager-Skills
- **定位**: 46 个产品经理技能模块

## 核心特性

覆盖 PM 核心技能：
- JTBD（Jobs To Be Done）分析
- 用户故事地图
- 优先级排序
- 竞品分析
- 发现访谈
- 定位声明

## 与生态的关系

- 与 [[agency-agents]] 互补：agency-agents 定义 PM 角色职责，本项目提供具体技能模块
- 属于 [[knowledge-as-code]] 趋势在产品管理领域的体现
