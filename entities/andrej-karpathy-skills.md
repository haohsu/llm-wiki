# Andrej Karpathy Skills

## 概述
GitHub 仓库 `forrestchang/andrej-karpathy-skills`，提供一个基于 Andrej Karpathy 观察的 Claude Code 指南文件（`CLAUDE.md`），旨在改进 Claude Code 的行为，解决 LLM 编码中的常见陷阱。

## 核心内容
- **来源**: 基于 Andrej Karpathy 在 X 上关于 LLM 编码陷阱的观察
- **形式**: 单个 `CLAUDE.md` 文件，包含四个核心原则
- **目标**: 改善 Claude Code 的假设管理、代码简洁性、修改精准度和目标驱动执行

## 四个原则
1. **Think Before Coding** — 明确假设，暴露不确定性，必要时提出异议
2. **Simplicity First** — 最小化代码，避免过度工程化
3. **Surgical Changes** — 精准修改，避免波及无关代码
4. **Goal-Driven Execution** — 通过测试和可验证标准驱动开发
