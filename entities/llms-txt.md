# llms.txt

- **目录站**: https://llmstxthub.com/
- **定位**: 让网站对 LLM 可读的标准
- **采用情况**: 900+ 网站已实现

## 核心特性

- 放在网站根目录的 Markdown 文件
- 用结构化方式描述网站内容
- 类比：`robots.txt` 给搜索引擎看，`llms.txt` 给大模型看
