# agency-agents

- **英文版**: https://github.com/msitarzewski/agency-agents
- **中文版**: https://github.com/jnMetaCode/agency-agents-zh
- **Stars**: 52k+
- **定位**: 147 个职业角色模板，覆盖 12 个部门

## 核心特性

- 147 个结构化 Markdown 角色文件
- 覆盖部门：前端、后端、DevOps、设计、产品、内容策略、营销、社区运营等 12 个
- 每个角色包含：职责定义、工作流程、交付物模板、质量标准

## 使用方式

将对应角色的 .md 文件丢给 Agent，Agent 就具备了该角色的专业能力。例如丢入"产品经理"模板，Agent 就能按标准 PM 流程工作。

## 与生态的关系

- 与 [[gstack]] 的区别见 gstack 页面
- 与 [[product-manager-skills]] 互补：agency-agents 定义角色职责，Product-Manager-Skills 提供具体技能模块
- 与 [[aicmo-marketing]] 互补：agency-agents 定义营销角色，AiCMO 提供具体营销 prompt
- 都属于 [[knowledge-as-code]] 趋势
