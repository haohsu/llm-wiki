# awesome-design-md

- **GitHub**: https://github.com/VoltAgent/awesome-design-md
- **Stars**: 4.3k
- **定位**: 55+ 大厂设计规范的 Markdown 集合

## 核心特性

- 55+ 家大厂设计系统提取为 Markdown 文件
- 覆盖品牌：Stripe、Figma、Notion 等
- 每个文件包含完整设计系统：配色、排版、组件、布局、阴影、响应式行为、设计禁区
- 丢进项目根目录，Agent 生成的 UI 就能匹配对应品牌风格

## 使用场景

当需要 Agent 生成特定品牌风格的 UI 时，将对应品牌的 .md 文件放入项目，Agent 会参考其设计规范进行输出。

## 与生态的关系

- 与 [[taste-skill]] 互补：awesome-design-md 提供品牌设计规范，taste-skill 提供风格控制旋钮
- 与 [[agent-instruction-standards]] 配合使用效果更好
- 属于 [[knowledge-as-code]] 趋势在设计领域的体现
