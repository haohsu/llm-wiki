# Public APIs

- **GitHub**: https://github.com/public-apis/public-apis
- **Stars**: 429,578
Public APIs 是一个社区维护的免费公共 API 集合，涵盖多个领域（股票、天气、航班等），方便开发者快速集成使用。

## 核心特性

- 社区维护，持续更新
- 覆盖金融、天气、交通等多个领域
- 429k+ stars，GitHub 上最受欢迎的项目之一

## 与生态的关系

- Agent 开发的重要基础设施：Agent 需要调用外部 API 获取实时数据
- 与 [[trading-agents]] 等需要市场数据的 Agent 项目互补
